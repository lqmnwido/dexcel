# Dexcel

Dexcel exports an entire database into one Excel file.

Each table becomes one sheet.

Supported databases:

- MySQL / MariaDB
- PostgreSQL
- SQLite
- Microsoft SQL Server
- Oracle

---

## Install

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/luqmanhafiz81/dexcel/main/install.sh | sh
```
